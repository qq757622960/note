Function Arguments 函数参数

形式及定义

当函数被定义时，会得到一个 "免费" 配送的参数，那就是 arguments 数组。它是一个 类似数组(array-like)

用途

构造一个大量数字相加的函数 

                        
    // 构造一个将大量的值相加的函数

    var sum = function () {
        var i, sum = 0;

        for (i = 0; i < arguments.length; i++) {
            sum += arguments[i];
        }

        return sum;
    }
