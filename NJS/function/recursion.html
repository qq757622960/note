Function recursion 递归

形式及定义

递归函数就是会直接或间接地调用自身的一种函数，它把一个问题分解为一组相似的子问题。一般来说，一个递归函数调用自身去解决它的子问题 

                        
    // 递归的形式
    var func = function () {
        func();
    };

用途

                        
    // 利用递归可以非常高效的操作树形结构

    var walkDom = function (node, func) {
        
        walkDom();
    };
