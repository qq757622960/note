The Apply invocation Pattern 构造器调用模式

Apply调用模式：形式及定义

apply 方法让我们构建一个参数数组传递给调用函数。它允许我们选择 this 的值。apply 接收 两个参数, 第一个是要绑定给 this 的值，第二个就是一个参数数组。 

                        
    func.apply(context, [arrayArguments]);

好处

暂无

用途

改变 this 指向

                        
    var obj1 = {
        name: 'mmx'
    };

    var obj2 = {
        name: 'xfj'
    };

    window.name = 'yy';

    var getName = function () {
        console.log(this.name);
    };

    // 输出 yy
    getName();  

    // 输出 mmx
    getName.apply(obj1);

    // 输出 xfj
    getName.apply(obj2);

借用其他对象的方法

                        
    Math.max.apply(null, [1, 5, 6]);
