The Function invocation Pattern 函数调用模式

函数调用模式：形式及定义

函数调用模式: 当一个函数不是一个对象的属性时，那么它就是被当做一个函数来调用的。 this 绑定到全局对象 

                        
    var func = function () {}
                        
                    
                        
    obj.double = function () {
        var that = this;

        var helper = function () {
            that.value = 123;
        }

        // 以函数形式调用 helper
        helper();
    }

    // 以方法形式调用 double
    obj.double();
