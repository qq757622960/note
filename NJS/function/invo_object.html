The Method invocation Pattern 方法调用模式

形式及定义

方法调用模式: 当一个函数被保存为对象的一个属性时, 我们称之为方法。 

                        
    var obj = {
        func: function () {}                                
    };

好处

可以把函数集中化，比如事件调用

用途

事件绑定 

                        
    $('body').on('click', '*[ys-event]', function (e) {
        var othis = $(this), methid = othis.attr('ys-event');
        events[methid] ? events[methid].apply(this, [othis, e]) : '';
    });

    // 事件表里面存放动作

    var events = {
        detail: function (othis, e) {
            console.log('详情');
        }
    };
