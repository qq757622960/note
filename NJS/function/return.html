Function Return 函数返回值

形式及定义

一个函数总会有一个返回值。如果没有指定返回值，则返回 undefined。如果函数调用时在前面加上了 new 前缀，且返回值不是一个对象，则返回 this (该新对象)。 

                        
    // 返回 undefined
    
    var func = function () {
        console.log(123);
    };


    // 返回 this

    var Person = function () {
        this.name = 'xfj';
    };
    var person = new Person();
    

    // 返回 return 的对象

    var Person = function () {
        this.name = 'xfj';

        return {
            name: 'mmx'
        }
    };
